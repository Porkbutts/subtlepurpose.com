# Jekyll Base

Base template for personal Jekyll websites


## Adding new gems
```
$ bundle add <gem>
```

## Run locally
```
# Development
$ ./bin/serve-dev.sh

# Production
$ ./bin/serve-prod.sh
```

## Collect tags from posts and create the layouts
```
$ ./bin/update_tags.rb
```
